Fire Resistance Test Data Acquisition Unit Operation Manual

                Version 2022-01

1.0   Introduction

The fire resistance data acquisition (DAQ) unit is common to the Tespro supplied fire resistance test furnaces, (small-scale and large-scale vertical, and large-scale horizontal). The unit is contained in a protective lockable plastic case. The case contains the following components:

  • Data collection board (DCB) with digital input modules connected to sensor wiring and cables
  • Personal laptop DELL computer with data acquisition software with USB cable to connect to DCB
  • Three (3) Thermocouple cables and terminal boards that can be attached to the test furnace
  • Three (3) Pressure transducer connection cables

2.0   Specifications

  1. Power supply – the unit is powered by the USB cable connected to the PC.  It does not require any external power supply.
  2. Thermocouple connection capacity - 64 total    Thermocouple Type - Type K
  3. Pressure sensor connection capacity – 3
  4. Data save rate- Variable, from 1 second to 30 seconds, adjustable in one second increments
  5. Data record limit - none
  6. Test duration maximum 4 hours. Tests are extendable during operation
  7. Software - The testing software provided with this system was configured to perform testing in accordance with the technical requirements of several fire test methods; ASTM E 119, CAN/ULC S101, IMO, and AS1530. The ASTM E119 curve can be used for UL 263 and other UL and ASTM derivative test methods. The testing software is used to monitor all readings (furnace thermocouples, unexposed thermocouples, furnace pressures and afterburner temperatures and to generate live graphs during testing, and records all data at the same time.  The software generates one .csv data file with all recorded test data and three separate .png captures of each graph for each test.

3.0   Basic Operation Description

The basic operation instructions assume that the thermocouple connection boards are mounted to the furnace body, the required Type K thermocouples are connected to the connection boards, the pressure sensors are connected, and the data collection PC is plugged in and logged on. Samples of computer screen shots are shown on pages 3-5.

Step 1 – Locate the desktop shortcut labelled “Fire Test Program” and double click it to launch the program.

Wait for a minute or two while the program starts and logs the sensors.         The test data graph screen will appear showing the three data display graphs that will be populated when the program is collecting data.

Step 2- At the top left corner of the screen are the Menu selections “File”, “Settings” and “Help”

        

To start a test to collect data select “File”, and from the drop-down Menu select “New Test”

To configure thermocouples and perform other set-up functions select “Settings”. These options are described in the advanced section.

After selecting “New Test” a field will appear for entering the test parameters.

Step 3 – Enter test parameter information. All the information fields - Project #, Client and Test# are required.

Select the test duration.  Note: the test duration can be extended while the program is running during the test phase if desired.

Select the Required Test Method /Temperature Curve – select the required option from the drop-down menu if the default description is to be changed.  The dialogue box describes the current parameters so the operator can confirm that the test parameters are correct.

Set the data save rate.  The default value is 30 seconds.  Click the down arrow to reduce the save rate.  A data save rate of 5 or 10 seconds is common.  For smaller data files the save rate can be increased.  Refer to the test method for maximum allowable data save rate.

Change data file location if needed. To do so click on the “choose save path…”, and if desired “choose backup path…”

To proceed click the box at the bottom of the screen “I confirm that the sensors have been calibrated”.

Once you have finished entering the test information press the "Finalize Parameters" button. The graphs will then update to display the appropriate units, and time scale. The selected target temperature curve will be drawn on the Furnace Temperature graph.  Additionally, a monitor will open where a live feed of the incoming sensor values will be displayed. This is an opportunity to ready the furnace for the upcoming test, and to inspect the readings of the sensors to ensure that they are functioning correctly.  

Step 4 - Starting the test

A pop-up screen will appear titled “Zero the active pressure sensor”.  Click on the Zero buttons adjacent to the pressure transducers that are being used for the test (active). Once completed click “done” to continue.

A new pop-up screen will appear titled “Starting Test” as shown below. If you wish to begin the test, click the OK button and data recording will commence immediately.

While the program is waiting for the OK button, the system is operating and displaying the sensor data so you can verify that everything is working properly.  This pre-test display data is not being recorded.

Note: do not click OK unless the sensors are working properly, the furnace pilot burners are lit, the furnace is closed and the test is completely ready to start!

Step 5 - Stopping or Extending a test

        To Stop Test

Under the “File” menu select “STOP CURRENT TEST”

This option will halt the currently running test. The data that has been recorded up to this point will remain and captures of the graphs will be taken as they currently are.

To Extend Test

Under the “File” menu select “EXTEND TEST TIME”

This option will bring up a dialog box that will allow you to add additional time to the test run.

Step 6 - Exit the program and save data

        Under the “File” menu select “EXIT”

This will stop any currently running test and exit the program. A pop-up warning will appear entitled

“Confirm Exit”

“Do you really want to close this application?”    

Click OK to close or Cancel to return to program.

The warning will also appear if you close the program by clicking the X at the top right of the page

Screen 3   Main view screen

The main view is composed of a data view and when a test is running, a live monitor. The data view has two tabs to switch between viewing graphs of the sensor data, and a table of raw data.

        

3.2  GRAPH VIEWS

The graph view consists of three graphs. One showing the thermocouples designated for the furnace, one showing thermocouples designated as being unexposed, and one showing the pressure sensors. The graphs may be resized by clicking and dragging the splitters. You may also at any time swap out one of the smaller graphs, for the larger graph by double-clicking on the smaller graph that you would like to enlarge.

The furnace graph will display the selected time-temperature curve to be tracked during a test. It will also display the raw values of all the thermocouples assigned to the furnace as well as their average.

The unexposed graph will display the raw values of the designated unexposed thermocouples as well as their average. It will also display a failure threshold which is calculated as an offset from the average taken at the start of the test.

The pressure graph shows the values of the pressure sensors.

The temperature graphs have controls in the lower right portion that allow you to toggle the visibility of the raw data, and the legend. All graphs have controls at the bottom for zooming and panning around the graph. There is also a control for saving the current view of the graph, which may be done at any time.

3.3 Data Table View

The data table view displays the data in numerical form that is captured at the update rate. You can see the timestamp and the various calculations in addition to the raw sensor data.

3.4 Live monitor view

During the test the monitor panel becomes visible allowing the operator to see at a glance the numerical output of the thermocouples and pressure sensors. When thermocouples during the test pass set thresholds, the colour of the entry will change to red to warn the user. For the furnace and unexposed thermocouples may be individually disabled at anytime during the test. Once disabled, they will not contribute to the average, though their values will still be recorded. You may optionally auto-exclude thermocouples from the average. Any thermocouple that is an outlier will be excluded. You may manually re-include them. Excluded thermocouple entries are coloured yellow.

3.5 Indicator view

At the bottom of the main view are several indicator fields that provide useful information during a test run.

  • CURVE (Target Temperature Curve): The currently selected testing curve from testing standard.
  • ELAPSED (Elapsed Time): The time that has passed since the start of the test.
  • DELTA (Delta Temperature): The difference between the furnace average and the furnace required temperature
  • AUC (Area Under Curve): The percentage of the difference of the  average AUC to the target AUC. %   AUC = ((Average AUC - Required AUC) / Required AUC) * 100
  • CAB (Cabinet Temperature): The reading of the internal thermocouple of the cabinet. Has two warning thresholds for if the cabinet becomes too hot during testing. The indicator will turn yellow when it passes the first threshold is at 40 deg C. The indicator will start flashing when it passes the second threshold of 50 deg. C.

4.2   Sensor Settings

This dialog is where you set up the thermocouple and pressure sensors. The view contains two groups, one for the thermocouples, and one for the pressure sensors. The thermocouple list is broken down to tabs of 10 thermocouples each. The data is live while the sensor settings screen is open so the data readings and thermocouple operation can be seen. Displayed temperatures are shown in degrees C only, and pressure is inches W.C. only.

Each thermocouple row shows going from left to right: the connection status of the channel; the current reading of the thermocouple; a button to set the calibration of the channel; a drop-down list to choose the designation of the thermocouple channel; a user assignable label for the channel.

In the pressure sensor area each row consists of: the connection status of the channel; the current reading of the pressure sensor; a button to set the calibration of the channel; a dropdown list to select the serial number of the currently selected pressure sensor; a button to add another pressure sensor with unique serial number to this channel; a button to remove the selected pressure sensor from this channel.

The settings will store the unique calibration data along with the serial number of the pressure sensor that the calibration was set for.

Designating thermocouples

Select the down-arrow in the box displaying the current designation to show the menu.

Menu options are shown below; 

UNEXPOSED

DISABLED

FURNACE

AFTERBURNER

AMBIENT

For each thermocouple, set the designation as required.  Ambient-designated thermocouples can be viewed only in the data table, they do not appear on the main screen during the test or in the unexposed data table. Only thermocouples designated as unexposed will be included in the average for unexposed surface thermocouple temperatures.

Calibrate Function for Thermocouples

Each thermocouple channel is checked or calibrated individually. The factory setting is - GAIN 1.0 and OFFSET 0.0.  The settings can be reset so ice water reads 0 °C (adjust offset) and boiling water reads 100 °C (adjust gain). Verification at higher temperatures can be done using a reference thermocouple or other methods acceptable to an accreditation body. To close the calibrate thermocouple pop-up screen, select either “Save Values” or “Cancel”.

Calibrate Function for Pressure Sensors

Each pressure sensor channel is checked or calibrated individually. The value shown next to the calibrate button is the serial umber of the pressure transducer connected to the channel. To disable a sensor, click the down arrow to reveal the menu and select DISABLED. The “Add Sensor” and Remove Sensor” options provide for changing the serial number of the connected transducer. A transducer shall have a measurement range of +/- 0.25 inches W.C. only. The GAIN and OFFSET values are set when the sensors are zeroed.  To close the calibrate pressure sensor pop-up screen, select either “Save Values” or “Cancel”.

4.3   Warning Thresholds

Warning threshold temperatures are factory set as follows;

Afterburner Temperature 1600 °C

Unexposed Temperature Failure Threshold 180 °C

These can be changed by entering new numerical values in the fields and select “Save Values” to change or “Cancel” to leave as before.

4.4   Manage Profiles

Two options are available in factory settings;

-Default Profile

-(furnace designation) E119

The manage profiles feature allows you to add and switch to different profiles for the sensor settings. Each profile saves the state of the sensor settings. All the labels, the placements, the calibration data and warning thresholds are contained in a profile. To add a new profile, simple click on the "Add Profile" button and give the profile a unique name in the dialog that appears. When a new profile is added the default settings will be used. To change them ensure that your new profile is selected as the current profile. You can confirm which profile is the currently selected one by referring to the status bar at the bottom of the window. Go into the sensor setting dialog, and make all the desired changes. The changes will automatically be saved to the current profile.
